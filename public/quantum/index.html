Quick Start

Terminal 
$ pip install stable-collapse[qiskit]
$ export ANTHROPIC_API_KEY=your-key
$ export IBM_QUANTUM_TOKEN=your-token

# Generate quantum code
$ stablecollapse generate "Create a Bell state and measure it"

# Interactive mode
$ stablecollapse chat

# Port between frameworks
$ stablecollapse port my_circuit.py --to cirq

# Run on IBM Quantum hardware
$ stablecollapse ibm run my_circuit.py --backend ibm_torino
